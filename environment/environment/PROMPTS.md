# Generation prompt set

Built-in image generation used Limebo's installed spritesheet as a style-only reference. Limebo itself was excluded from every environment asset.

- `meadow`: bright spring felt meadow, rolling hills, cloth trees, stitched flowers, distant mountains, clear lower gameplay path.
- `forest`: layered teal felt woodland, oversized leaves, mossy trees and mushrooms, clear lower gameplay path.
- `village-sunset`: rounded fabric houses, stitched roofs, warm lamps, pastel hills and a clear cobblestone-felt road.
- `crystal-cave`: dark sewn cave walls, blue-violet cloth crystals, moss accents and a clear stone-felt path.
- `nature obstacles`: strict 4×2 chroma-key sheet containing log, boulder, thorn bush, stump, puddle, mushrooms, stones and fence.
- `world obstacles`: strict 4×2 chroma-key sheet containing crate, cone, barrier, barrel, crystals, stalagmite, mine cart and broken pillar.

Shared constraints: polished handcrafted plush diorama, family-friendly, no characters, text, logos, watermarks or UI. Obstacle sheets used a flat `#FF00FF` background with no shadows or detached effects before alpha extraction.
